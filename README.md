# 電子木魚 - 積功德網頁程式

一個簡單有趣的網頁應用程式，透過點擊木魚來積累功德值。

## 功能特色

- 🎯 **點擊積德**：每次點擊木魚累積功德值
- 💾 **自動儲存**：功德值自動保存在瀏覽器中
- 🎵 **音效回饋**：點擊時播放木魚聲音
- ✨ **視覺效果**：漂浮文字與漣漪動畫
- 🏆 **成就系統**：達到特定功德值時觸發成就
- ⌨️ **鍵盤支援**：按空白鍵快速敲擊
- 📱 **響應式設計**：支援各種螢幕尺寸
- 🔄 **重置功能**：可重新開始累積功德

## 使用方式

1. 直接在瀏覽器開啟 `muyu.html` 檔案
2. 點擊木魚或按空白鍵開始積累功德
3. 功德值會自動儲存，下次開啟時繼續累積

## 成就里程碑

- 10 功德 - 初心者
- 50 功德 - 虔誠信徒
- 100 功德 - 功德圓滿
- 500 功德 - 功德無量
- 1000 功德 - 功德大師
- 5000 功德 - 功德至尊

## 技術實現

- 純 HTML/CSS/JavaScript 實現
- 使用 Web Audio API 產生音效
- LocalStorage 儲存進度
- CSS 動畫效果
